# Tailwind → Figma mapping

## Spacing
- p-1 → 4px
- p-2 → 8px
- p-4 → 16px
- p-6 → 24px
- p-8 → 32px

## Gap
- gap-2 → 8px
- gap-4 → 16px
- gap-6 → 24px

## Border radius
- rounded-sm → 4px
- rounded-md → 8px
- rounded-lg → 12px
- rounded-xl → 16px

## Typography
- text-sm → body small
- text-base → body
- text-lg → subtitle
- text-xl → heading small
- text-3xl → heading large

## Colors
- bg-* → background tokens
- text-* → text tokens
- border-* → border tokens

Convert to semantic tokens when possible.
