# Inspiration Engine

This system extracts visual design intelligence from inspiration images.

## Flow

1. asset-decomposition.md
2. cross-image-consolidation.md
3. creative-synthesis.md
4. approval-flow.md

## Rules

- Images are analyzed individually first
- No averaging — only curation
- No tokens before approval
- No UI generation before approval
- Treat output as perceptual, not technical

## Output

- Creative Style Profile
- Design Principles
- Layout Patterns
- Anti-patterns
