# Final Visual Inspection

Run this AFTER UI is created (Figma or code).

## Check 1 — Layout integrity
- no overlapping elements
- spacing is consistent
- alignment is clean

## Check 2 — Hierarchy
- clear primary CTA
- readable headings
- proper visual flow

## Check 3 — Animation quality
- animations are not excessive
- durations feel natural
- no jitter or broken transitions
- parallax does not affect readability

## Check 4 — Responsiveness
- works on mobile
- no overflow issues
- spacing adapts properly

## Check 5 — Consistency
- components behave consistently
- buttons follow same pattern
- cards follow same structure

## Fix behavior

If issues found:
- fix immediately
- do not leave broken UI
- prefer removing animation over breaking UX
