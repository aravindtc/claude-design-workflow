# Output Format

After completing all variations, report in this exact structure.

---

## 1. Source inspected

- **Name:** frame name
- **ID:** node id
- **Dimensions:** width × height
- **Diagnosis:** 2–4 sentences on what works, what is weak, what was preserved, what changed

---

## 2. Variations created

For each variation:

- **Name:** variation name
- **ID:** node id
- **Position:** x, y coordinates
- **Concept:** one sentence describing the design idea
- **Key changes:** bullet list of what moved, changed, or was added/removed
- **Why it works:** one sentence on the design rationale

---

## 3. Quality check

| Variation | Overlap | Legibility | Hierarchy | Alignment/Spacing |
|---|---|---|---|---|
| Name | ✅ Pass / ❌ Fail | ✅ Pass / ❌ Fail | ✅ Pass / ❌ Fail | ✅ Pass / ❌ Fail |

If any check is Fail, fix it before concluding and update the table.
