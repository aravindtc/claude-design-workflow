# Design Principles

## Design philosophy

Every variation must have:
- A clear communication goal
- A visible information hierarchy
- A coherent reading path
- Intentional spacing
- Strong alignment
- Balanced positive/negative space
- Believable image/text relationship
- Consistency of typography, color logic, and component behavior

A variation is not "different" unless it changes the composition logic.
A variation is not "good" unless it improves clarity, emphasis, rhythm, or emotional tone.

---

## 1. Preserve intent before changing form

Infer from the source:
- Likely audience
- Core message
- Primary CTA
- Emotional tone
- Dominant content priority

Protect these unless the user asks to change them.

## 2. Use hierarchy, not decoration

Make the most important content the most visually prominent through:
- Position
- Scale
- Contrast
- Whitespace
- Grouping

Do not add accents that do not support hierarchy.

## 3. Design on a grid

Use a consistent underlying grid and alignment system.
Elements may break the grid only if the break is clearly intentional and improves emphasis.

## 4. Control spacing rhythm

Use consistent margins, gutters, and internal spacing.
Prefer fewer, stronger spacing relationships over many inconsistent ones.

## 5. Respect reading flow

For Latin-script content, default reading flow is top-left to bottom-right unless the composition strongly justifies another path.

## 6. Maintain text legibility

Never place text where contrast, busy imagery, or proximity harms readability.
If using overlay text, create sufficient contrast with overlays, containers, cropping, or repositioning.

## 7. Keep visual relationships believable

Image crop, subject placement, text block placement, CTA location, and accent shapes should feel compositionally related — not independently moved.

## 8. Use color purposefully

A color variation is not just hue replacement.
It should preserve or improve:
- Contrast
- Tone
- Emphasis
- Brand feeling
- CTA salience

## 9. Avoid common automation mistakes

Never:
- Let text overlap
- Let bounding boxes collide
- Create accidental tangencies
- Trap text against edges
- Create tiny orphan accents
- Leave elements visually floating without alignment logic
- Squash or stretch images unnaturally
- Create equal emphasis for everything
- Create dead zones or congested zones without intent

## 10. Exercise restraint

A good variation usually changes one primary design idea:
- Layout structure
- Image strategy
- Emphasis model
- Tone via color or material

Do not change everything at once unless explicitly asked.
