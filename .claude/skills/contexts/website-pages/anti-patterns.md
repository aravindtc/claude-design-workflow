# Website Page Anti-Patterns

_Stub — populate when working on website page variations._

## Common mistakes

- Hero that uses jargon instead of clear outcome language
- Section headings that label instead of sell ("Features" vs "Everything you need to...")
- Inconsistent spacing between sections (some sections 80px padding, others 40px)
- Too many font sizes creating visual noise instead of hierarchy
- CTA buttons that blend into the design rather than standing out
- Images used as decoration without supporting the content they sit beside
