# Website Page Heuristics

## Key principles

- Navigation should be immediately recognizable and consistent
- Page should be scannable — users read in F-pattern or Z-pattern at first glance
- Section headers should communicate value, not just label content
- White space is structure — not emptiness
- Brand expression should be consistent but not overpowering
