# App Screen Anti-Patterns

_Stub — populate when working on app screen variations._

## Common mistakes

- Primary CTA in the top half of the screen (unreachable thumb zone)
- Icon-only navigation without labels (forces memorization)
- Text smaller than 14px in body content
- Touch targets smaller than 44×44pt
- No visual distinction between interactive and non-interactive elements
- Inconsistent component usage across screens in the same flow
- Overloaded screens trying to show everything at once
