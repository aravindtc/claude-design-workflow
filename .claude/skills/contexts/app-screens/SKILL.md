## App screen context

App screens are task-oriented UI surfaces.

Design decisions must support:
- Clear task completion path
- Consistent component behavior
- Legible at system font sizes (minimum 16px body)
- Accessible tap targets (minimum 44×44pt)
- Navigation clarity — user always knows where they are

Common frame sizes: 390×844 (iPhone 14), 360×800 (Android), 1024×1366 (iPad)

Load context files for screen-type specific guidance.
