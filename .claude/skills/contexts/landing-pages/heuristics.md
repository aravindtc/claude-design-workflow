# Landing Page Heuristics

_Stub — populate when working on landing page variations._

## Key principles

- Above the fold must answer: what is this, who is it for, what do I do next
- One primary CTA per page section — do not split attention
- Value propositions need visual separation, not walls of text
- Social proof (logos, quotes, numbers) should be close to the decision point
- Mobile: stack sections vertically, increase tap target sizes, reduce copy density

## Hierarchy model

1. Hero: headline + subheading + primary CTA
2. Proof bar: logos or stats (trust signal, low friction)
3. Value props: 3 columns or alternating rows
4. Feature deep-dive: image + copy pairs
5. Testimonials: 1–3 strong quotes with attribution
6. Final CTA: repeat primary action with urgency framing

## Spacing

- Generous section padding: 80–120px top/bottom at desktop
- Consistent column gutter: 24–32px
- Text max-width: 680px for body, 800px for headlines
