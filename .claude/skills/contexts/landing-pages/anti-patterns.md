# Landing Page Anti-Patterns

_Stub — populate when working on landing page variations._

## Common mistakes

- Multiple competing CTAs above the fold
- Hero headline that describes features instead of the outcome for the user
- No visual hierarchy between headline, subheading, and body text
- Social proof buried below the fold with no early trust signal
- Form fields appearing before value has been established
- Dense paragraph text with no visual breaks (bullets, icons, whitespace)
- Navigation with too many links distracting from the conversion goal
