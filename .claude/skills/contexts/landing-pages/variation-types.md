# Landing Page Variation Types

_Stub — populate when working on landing page variations._

## Hero variations
- Centered headline + full-bleed background image
- Left-aligned headline + right-side product/app screenshot
- Bold typographic hero — no image, strong headline
- Split-screen: copy left, visual proof right

## Section flow variations
- Long-form scroll (all sections visible, heavy proof)
- Short-form (hero + 3 value props + CTA — minimal friction)
- Story-driven (sequential narrative sections)

## CTA model variations
- Single persistent CTA (same action repeated at each section)
- Progressive disclosure (soft CTA early, hard CTA after proof)
- Urgency model (countdown, limited spots, deadline framing)
