# Typography Patterns

Guidelines for typographic decisions across all design variation contexts.

## Scale

Use a clear type scale with distinct size jumps — not incremental steps that look accidental.

Suggested scale for social posts (1080px frames):
- Display / hero headline: 72–120px
- Section headline: 48–72px
- Subheading: 28–40px
- Body / supporting text: 20–28px
- Label / caption: 14–18px

Suggested scale for web (1440px):
- Display: 56–80px
- H1: 40–56px
- H2: 28–40px
- H3: 20–28px
- Body: 16–18px
- Caption: 12–14px

## Weight

- Use weight contrast to reinforce hierarchy (bold headline, regular body)
- Avoid using more than 2–3 weights in a single frame
- Do not use light weight text on complex or patterned backgrounds

## Alignment

- Left-aligned text is easiest to read for long-form content
- Center-aligned works for short headlines, centered layouts, and hero moments
- Right-aligned works for captions beside left-anchored images
- Never mix alignments without clear compositional reason

## Line length

- Optimal line length: 55–75 characters for body text
- Headlines can be shorter (2–6 words per line for maximum punch)
- Avoid single-word lines unless intentional typographic emphasis

## Spacing

- Line height: 1.2–1.4 for headlines, 1.5–1.7 for body text
- Letter spacing: tight for large display text, default for body
- Paragraph spacing: 0.75–1× the font size

## Legibility rules

- Minimum font size: 14px for any text a user must read
- Never set white text on a light background without sufficient contrast
- Never set dark text on a dark background without sufficient contrast
- Avoid italic or condensed faces for long body text
