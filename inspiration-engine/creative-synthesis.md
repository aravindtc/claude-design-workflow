# Creative Synthesis

Generate new design directions by combining and reinterpreting consolidated inspiration patterns.

## Principles

- **Define variation axes before designing** — Decide what changes between variants (e.g., tone, layout, color) and what stays constant
- **Prefer 3-5 strong variants over many weak ones** — Each variant should have a clear reason to exist
- **Keep hierarchy clear** — Every variant must communicate the same core message, even if the visual approach differs
- **Do not randomly restyle** — Each direction should trace back to specific patterns from the consolidation phase

## Output

A set of distinct creative directions, each with:
- A short name and rationale
- The variation axes it explores
- Key design decisions and their inspiration sources
