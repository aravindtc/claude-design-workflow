# Asset Decomposition

Break down each reference image into its constituent design elements.

## Elements to Extract

- **Layout** — Grid structure, spacing, alignment, content hierarchy
- **Typography** — Font styles, sizes, weights, pairings, text treatments
- **Color** — Palette, contrast ratios, color relationships, gradients
- **Imagery** — Photo style, illustration style, iconography, textures
- **Composition** — Visual flow, focal points, whitespace usage, balance
- **Effects** — Shadows, overlays, blur, borders, motion cues

## Output

For each reference image, produce a structured breakdown of the elements above, noting what makes each element distinctive or effective.
