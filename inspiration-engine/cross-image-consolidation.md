# Cross-Image Consolidation

Find patterns and contrasts across multiple reference images to identify reusable design strategies.

## Process

1. **Pattern Detection** — Identify recurring elements across references (shared color approaches, similar layouts, common typographic choices)
2. **Contrast Mapping** — Note where references diverge and what makes each approach distinct
3. **Strength Extraction** — Pull the strongest element from each reference rather than averaging them together
4. **Theme Clustering** — Group related patterns into coherent design themes that could inform variations

## Output

A consolidated set of design patterns, organized by theme, with clear attribution to which reference(s) each pattern derives from.
