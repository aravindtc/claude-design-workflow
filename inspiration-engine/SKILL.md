# Inspiration Engine

Skill for extracting design inspiration from reference images and synthesizing creative directions for social media design variations.

## Workflow

1. **Asset Decomposition** — Break down each reference image into its constituent design elements
2. **Cross-Image Consolidation** — Find patterns and contrasts across multiple references
3. **Creative Synthesis** — Generate new design directions informed by the consolidated inspiration
4. **Approval Flow** — Present options and iterate based on feedback
