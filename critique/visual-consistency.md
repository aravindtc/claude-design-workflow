# Visual Consistency Checklist

Use this checklist to evaluate whether a set of design variations maintains visual consistency across the series while still being meaningfully distinct.

---

## Color Consistency

- [ ] Palette is limited to 2–3 hues per frame, shared across all variations
- [ ] Brand accent color is used in the same role (emphasis/CTA) across variations
- [ ] Background tones stay within the same family (all warm neutrals or all cool neutrals)
- [ ] Text colors remain neutral and high-contrast — not swapped between variations without reason
- [ ] No variation introduces a color that contradicts the palette logic of the set
- [ ] Contrast ratios meet minimums (4.5:1 body text, 3:1 large text/UI) in every variation

## Typography Consistency

- [ ] Same typeface(s) used across all variations
- [ ] Font weight assignments are stable (e.g., bold = headline, regular = body)
- [ ] Size scale is coherent — relative proportions between headline, subhead, and body stay intact
- [ ] Line height and letter spacing do not shift arbitrarily between variations
- [ ] No variation introduces a new font weight or style without clear intent

## Spacing and Alignment

- [ ] Margins and padding follow a consistent base unit across variations
- [ ] Alignment logic is stable within each variation (left-aligned stays left-aligned, centered stays centered)
- [ ] Gutters between elements are uniform — no accidental tight/loose spots
- [ ] Frame edges have consistent inset padding across the set

## Element Treatment

- [ ] CTA button style (shape, fill, border, text style) is consistent across variations
- [ ] Image treatment (rounded corners, overlays, crop ratios) does not shift randomly
- [ ] Decorative elements (lines, shapes, badges) follow the same visual language
- [ ] Icon style (outline vs filled, size, stroke weight) is uniform

## Brand Coherence

- [ ] All variations feel like they belong to the same campaign or brand moment
- [ ] Tone of visual language matches the message tone (e.g., bold visuals for bold copy)
- [ ] No single variation looks like it was designed for a different brand

---

## When inconsistency is intentional

Not all variation requires uniformity. Acceptable divergences:

- **Layout structure** — variations should explore different compositions
- **Emphasis shift** — one variation may lead with image, another with headline
- **Mood range** — a dark variant vs a light variant is a valid axis

The key test: if you removed all text, would the variations still feel like a cohesive set?
